## running the project

1. Clone the repository
```bash
# git clone
git clone URL_ADDRESS.com/your-username/collaboration-demo.git
```

2. Install dependencies
```bash
# pnpm
pnpm install
```
3. Run the project
```bash
# pnpm
pnpm start
```
